# hse-recipes

Система управления рецептами — консольное приложение для создания блюд, управления ингредиентами и генерации списка покупок.

## Установка

git clone https://github.com/Erik180/hse-recipes.git
cd hse-recipes
pip install -r requirements.txt

## Использование

Запуск тестов:

pytest
